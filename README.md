# Kaizen Inn

O Kaizen Inn é um sistema web de gestão hoteleira (Back-office) desenvolvido para a disciplina de Engenharia de Software III (4º Sem. FATEC). 

O projeto tem como foco digitalizar as operações diárias de um hotel, controlando o cadastro de hóspedes, quartos, fluxo de reservas, pagamentos e a rotina de check-in/check-out na recepção.
